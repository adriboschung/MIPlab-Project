# MIPlab-Project
Repository containing the code and other files related to my project at MIPlab, focusing on using iEEG data to follow the experiment done in https://pubmed.ncbi.nlm.nih.gov/38844456/ and analyzing/comparing the results.

In the folder "figures", you can find relevant figures showing the electrode placement and the regions we focused on, the PSD analysis, the envelopes analysis and other interesting exploration.

In the folder "videos", you will find the activity (ERP) of the electrodes for different conditions masked by the T1 brain mask.

The main notebook containing all the relevant code can be found in PAT_3455_processing.ipynb

Notes:
    - One big limitation of the project is that it was done on a single patient due to time constraints, which inevitably undermines the quality of the results and of the statistical analysis. A follow-up to this project would be to extend it to all the available data.
